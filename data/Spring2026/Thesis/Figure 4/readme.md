__Figure 4.__ All 21 possible 3-gene, 4-edge networks with the type of motif labeled and grouped by type. Blue, blunt arrows indicate repression and red, pointed arrows indicate activation with thicker arrows representing the larger magnitude weight (-2, 2) and thinner arrows representing the smaller magnitude weight (-1, 1). The number of each network corresponds to the rank in Table 13. 

`.png` files are the outputs from GRNsight that were used to make the figure. To get these outputs, any of the input files for the corresponding network can be run in GRNsight. 
_(If error occurs, download the file and delete ~10 rows below the last gene in the degradation_rates and production_rates sheets. Some of the sheets have lingering data that causes errors in GRNsight.)_

The numbers of the folder and .png file names follow the ranking in Table 13.
